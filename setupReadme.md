# HOW TO INSTALL ??? ?? ?

```
cd 到项目根目录
./installAll
```
 ```

```
如果不能安装,看这里
```
1. 先安装go环境. 去官网下载
然后go env检查是否安装好

2. 把PATH配置到.zshrc或者.bashrc(用哪个配哪个)
这个path就是go env里面的"GOPATH"
export PATH="/Users/xxx/go/bin:$PATH"
```
